# Web to Figma - Figma Plugin

Import web pages as pixel-perfect, editable Figma designs.

## Installation

1. Install dependencies:
npm install

2. Build the plugin:
npm run build

3. In Figma Desktop:
   - Plugins → Development → Import plugin from manifest
   - Select manifest.json from this directory

## Usage

1. Run plugin: Plugins → Development → Web to Figma Importer
2. Load JSON from Chrome extension
3. Configure import options
4. Click "Import to Figma"

## Features

- Main page frame with pixel-perfect layout
- Interactive variants frame
- Components library
- Design system page
- Auto Layout support
- Color & text styles
- Vector assets
